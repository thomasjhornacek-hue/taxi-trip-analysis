# Exploratory Taxi Trip Analysis with Hive on Hadoop

An exploratory analysis of a public yellow taxi trip dataset using HiveQL on Hadoop. Built as an applied project for the Johns Hopkins Big Data Analytics certificate.

## Overview

This project demonstrates querying a large trip dataset in a distributed data warehouse. Using HiveQL on Hadoop, it explores trip patterns across time, distance, fare, and revenue dimensions, illustrating how to work with data at warehouse scale rather than in memory.

## Data

A public yellow taxi trip dataset (provided with the course) with more than fifteen columns covering pickup and dropoff times, distances, fares, and related trip attributes.

## Approach

Loaded the dataset into Hive on a Hadoop environment, then ran a series of HiveQL queries across trip, time, and revenue dimensions. The analysis covered totals, averages, and time-of-day patterns, including identifying the hour with the highest average revenue.

## Tools

Hive, HiveQL, Hadoop, distributed data warehousing, SQL.
